# student_management_ostad_assignment11
Create a Student Management System using JavaScript involves several key features such as storing student data, handling forms, and manipulating the DOM.  Create a simple version of this system.

1. Add a Student: Input student details (name, age, grade, etc.)

2. Display the Student List: View the list of all students.

3. Update a Student: Edit student details.

4. Delete a Student: Remove a student from the list.

Key Features:

· HTML for structure.

· CSS for styling.

· JavaScript for functionality.
